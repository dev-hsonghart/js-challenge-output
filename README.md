# js-challenge-output - 모멘텀
노마드코드. 자바스크립트. 챌린지. 최종결과물


## 개요
노마드코더에서 자바스크립트 강의와 2주 챌린지를 통해 작업한 모멘텀입니다.

## 기능
- 위치 정보와 API를 통해 날씨, 지역, 온도 불러오기
- 로컬스토리지 사용하기
- 사용자 이름과 투두리스트 목록 저장하기
- 현재 시간 실시간 출력
- 배경이미지와 인삿말 랜덤 출력

## 업데이트
- 기능을 추가하거나 UX 수정을 할 계획입니다.
